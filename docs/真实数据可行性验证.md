# 真实数据可行性验证

## 验证设置

- 两组数据各自动选取一个 `512 × 768` RGB ROI；
- 输出 691–2518 nm、367 波段、float32 ENVI BIP；
- 低分辨率共同分析网格均为 `148 × 82`；
- 光谱子空间秩为 10；
- 两个输出立方体均为 550.5 MiB；
- 单组完整运行约 18–25 秒（当前工作站、含约 550 MiB ENVI 写出）。

## 结果

| 指标 | 3DSSZ | ZKH3 |
|---|---:|---:|
| 质量状态 | passed | passed |
| NIR 配准结构相关 | 0.6340 | 0.5130 |
| SWIR 配准结构相关 | 0.6366 | 0.5342 |
| 重叠区 RMSE（校准前） | 0.18755 | 0.15809 |
| 重叠区 RMSE（校准后） | 0.12809 | 0.13029 |
| 系数回投影 RMSE | 0.07401 | 0.05111 |
| 连续立方体回投影 RMSE | 0.01243 | 0.00863 |
| 连续立方体回投影 SAM | 1.296° | 0.869° |
| 连续立方体平均波段相关 | 0.9927 | 0.9909 |
| NIR 原传感器重投影 RMSE | 0.04015 | 0.06960 |
| SWIR 校准后重投影 RMSE | 0.05995 | 0.04162 |
| 低秩累计解释方差 | 0.99897 | 0.99931 |

3DSSZ 的初始 PSF 在自一致检查中缩减为 0.75 倍；ZKH3 缩减为 0.5 倍。ZKH3 在无安全门时出现明显反卷积振铃，安全回退将系数 RMSE 从 0.1325 降到 0.0511，并消除了大部分可疑细碎纹理。

## 结构验证

两组最终输出均满足：

- ENVI 头尺寸与二进制字节数一致；
- 立方体形状与 `manifest.json` 一致；
- 波长严格递增，首末波长为 691 nm 和 2518 nm；
- 抽样值全部有限且非负；
- 每个运行目录可由 `geocorefusion validate` 独立重新验证。

## 结论与限制

本阶段已经证明：两套真实岩心扫描可在统一代码下完成自动配准、重叠区波长相关校准、连续谱轴生成、RGB 系数边界引导、回投影安全控制和标准 ENVI 写出。

这些结果支持“技术路线可行”，但不能替代论文中的独立高分辨率真值。特别是原传感器重投影 SAM 仍受 SRF 未知、局部错位、阴影和坏波段影响；下一阶段需要用 HR-HSI、点光谱或矿物标签判断新增空间细节的真实性，并开展正式对比与消融。
