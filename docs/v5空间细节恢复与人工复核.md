# v5 空间细节恢复与人工复核

## 问题诊断

v4 的主要空间细节来自一个对全部 367 个波段共用的乘性增益。该方法能够增强亮度纹理，但不会改变低秩材料系数的高分辨率结构，因此容易出现“颗粒增强了，岩块边界和内部结构仍然偏平滑”的结果。继续增大增益还会同步放大传感器噪声和残余配准误差。

原始 M0-1 classical 方法更清晰的关键不是单纯锐化强度更大，而是同时使用了低秩系数细节注入和立方体级乘性/加性混合细节。v5 将这一思想改造成受观测约束的实现。

## v5 方法

1. 将 RGB 转换为亮度、红绿差和蓝绿差三个互补结构特征，并分解为低频结构与多尺度高频残差。
2. 对每个低秩材料系数分别拟合岭回归，自动确定 RGB 高频细节的注入方向、符号和可靠度；不再向所有系数注入相同符号的纹理。
3. 使用 HSI 材料边界支持和 RGB 有效像元置信度门控细节，降低背景、过暗区、饱和区和不受 HSI 支持的孤立 RGB 纹理影响。
4. 将系数细节投影到传感器退化模型的观测近零空间，然后执行低分辨率系数回投影，保持 NIR/SWIR 观测一致性。
5. 在重建阶段保留较弱的全波段共用增益，并增加一个波段自适应加性细节项。加性项主要补偿暗反射波段中纯乘性方法注入不足的问题，同样使用近零空间约束。

## 真实数据候选结果

| 数据 | 推荐候选 | 系数 RMSE：v4 → v5 | 连续立方体 RMSE：v4 → v5 | SAM：v4 → v5 | 900 nm 高频能量：v4 → v5 |
|---|---|---:|---:|---:|---:|
| 3DSSZ | Structure-strong v5 | 0.00753 → 0.00665 | 0.00437 → 0.00449 | 0.9076° → 0.9050° | 0.0863 → 0.0986 |
| ZKH3 | Hybrid-sharp v5 | 0.04903 → 0.04050 | 0.01017 → 0.00909 | 0.6425° → 0.5570° | 0.0763 → 0.0978 |

3DSSZ 的连续立方体 RMSE 有极小幅上升，但仍远低于 0.08 安全阈值，同时系数 RMSE、SAM 和空间结构均改善。ZKH3 的系数 RMSE、连续立方体 RMSE 和 SAM 均明显改善。高频能量只用于比较细节恢复强度，不能单独证明高分辨率 SWIR 真值。

## 当前处理决定

人工确认前不重写完整 367 波段高分辨率立方体，以避免在参数尚需视觉选择时生成两份约 2.3 GB 的临时结果。确认推荐候选后运行：

```powershell
python -m geocorefusion.cli run configs\3dssz_roi_fusion_v5.yaml
python -m geocorefusion.cli run configs\zkh3_roi_fusion_v5.yaml
```

完整 v5 输出会额外保存 `metrics/spatial_additive_detail.hdr/.dat`，并在质量报告中记录乘性增益和加性细节的低分辨率残差。
