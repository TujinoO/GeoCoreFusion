# 技术路线实现

## 1. 本阶段问题定义

目标是在 RGB 空间网格上生成 691–2518 nm 连续高光谱立方体。原始观测满足近似模型：

\[
Y_k=S_k B_k W_k R_k(X)+N_k,\quad k\in\{NIR,SWIR\}
\]

其中 `W` 为推扫几何形变，`B/S` 为空间模糊与采样，`R` 为传感器光谱响应。当前代码将该完整联合反演拆成可运行、可解释的阶段式自监督原型，同时保留未来替换为可学习模块的接口。

## 2. 配准

- RGB、NIR、SWIR 分别构建亮度、光谱对比和梯度组合的模态稳定结构图。
- 原始尺寸比例先提供各向异性的初始尺度，避免让 ECC 自由拟合不合理倍率。
- 同时尝试平移、欧氏和仿射模型，只接受对角尺度与交叉项满足推扫物理范围的候选。
- 沿扫描方向分段估计残余位移，以 MAD 过滤异常段，并把全局中位位移吸收到主变换中。
- 最终模型直接提供 `RGB 坐标 → 传感器坐标` 映射，低分辨率分析网格通过原始 ENVI memmap 按需采样。

当前实现仍是“先估计配准、再融合”。下一论文阶段可把变换参数改为可微参数，由观测重建损失联合微调。

## 3. 波长相关光谱统一

NIR 与 SWIR 在约 978–1482 nm 重叠。实现步骤如下：

1. 将两传感器采样到同一 RGB 几何分析网格。
2. 在 SWIR 重叠波长处插值 NIR 光谱。
3. 对每个重叠波长使用分位数初始化和 IRLS 稳健线性回归，估计 `NIR ≈ gain(λ) × SWIR + offset(λ)`。
4. 对 `gain(λ)` 与 `offset(λ)` 进行 Savitzky–Golay 平滑，并向 SWIR 全谱段延拓。
5. 估计逐波段空间高频噪声，以逆噪声方差作为可靠性。
6. 输出 691–2518 nm、5 nm 主间隔的统一谱轴；末端显式加入 2518 nm，因此共 367 波段。
7. 重叠区同时使用余弦渐变和波段可靠性加权，避免重复保留 553 个原始波段。

1400 nm 和 1900 nm 附近水吸收/低 SNR 波段在 `band_metadata.csv` 中标记为低置信度。

## 4. 光谱安全的 RGB 引导

统一低分辨率立方体分解为：

\[
X_{LR}=\mu+A_{LR}E
\]

其中 `E` 是仅由 NIR/SWIR 决定的光谱基，`A` 是材料系数场。RGB 不直接改写光谱基或逐波段辐射值。

高分辨率系数场通过以下目标迭代：

- 双三次初值与低分辨率系数提供光谱材料锚点；
- RGB 经双边平滑后生成边界权重，抑制细碎可见光纹理；
- 只有 RGB 边缘落在低分辨率材料边界膨胀支持区时，guided anchor 才允许边界迁移；
- 各向异性扩散在 RGB 同质区域内平滑系数，在 RGB 边界处停止；
- 每轮把系数经 PSF 和面积下采样回原分析网格，利用残差回投影；
- 逐分量限幅与系数物理范围裁剪防止反卷积振铃和光谱幻觉。

最终高分辨率立方体由 `X_HR=μ+A_HR E` 分块重建并写入 ENVI BIP 文件。

## 5. 退化模型与安全回退

代码通过 RGB 与高光谱结构的边缘一致性搜索方向不同的高斯 PSF。当跨模态相关不足以辨识 PSF 时，使用与空间倍率匹配的保守物理默认值。

若所选 PSF 下的系数回投影 RMSE 超过阈值，系统自动按 `1.0、0.75、0.5、0.25` 缩减 PSF 并重新融合，选择满足观测一致性的最强可接受退化。该机制在 ZKH3 真实 ROI 上把有振铃风险的 PSF 自动退回到 0.5 倍。

## 6. 不确定性

空间不确定性综合：

- 系数回投影残差；
- RGB 暗区、饱和区和高色差区域的低置信度；
- RGB 边界注入位置。

波段不确定性综合 NIR/SWIR 噪声可靠性和重叠区跨传感器差异。当前属于相对不确定性评分，下一阶段可扩展为异方差似然和标定曲线。

- 人工地质编录和矿物掩膜的下游任务评价；
