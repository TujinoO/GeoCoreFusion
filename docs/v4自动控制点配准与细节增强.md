# v4 自动控制点配准与 RGB 细节增强

## 改进目标

v3 的主要残差表现为两类：

1. RGB 高频纹理注入位置与 NIR/SWIR 低频结构不完全一致，产生双边缘；
2. NIR 与 SWIR 之间仍有局部非刚性误差，使 2200/1650/900 nm 伪彩色出现蓝黄分离。

v4 参考 `Geocore_M0&1_Preprocessing/modules/M0-1_image_fusion` 的自动控制点思想，在已有 ECC、重叠波段约束和分段横向校正之后增加局部配准。

## 自动控制点流程

1. 在最终融合所使用的低分辨率观测网格上构造跨模态结构图；
2. 按规则网格生成候选控制点；
3. 在局部窗口内搜索最大归一化相关位置，并用响应曲面二次拟合获得亚像素位置；
4. 执行正向—反向一致性检查；
5. 按匹配分数、峰值间隔、反向误差、位移 MAD 和最大位移剔除异常点；
6. 用带置信度权重的近邻 IDW 插值得到平滑二维位移场；
7. 自动测试 `1.0/0.75/0.5/0.25/0` 五档强度，并检查 NIR–SWIR 一致性与形变 Jacobian；
8. 分别执行 SWIR→RGB、NIR→RGB、NIR→SWIR 重叠波段以及共同 HSI→RGB 四个候选阶段，无收益阶段自动选择 0。

该流程不会为了提高单一相关系数而强制使用全部局部形变。

## RGB 细节增强

v4 继续对全部 367 个波段使用同一个乘性空间增益，因此不会对不同波段注入不同纹理。相对 v3 的变化包括：

- 提高微纹理保留权重；
- 将细节归一化分位数从 98% 调整为 95%；
- 使用双曲正切软限制，减少硬裁剪；
- 增加纹理区的最低注入权重；
- 增加观测近零空间投影次数，使增强后影像退化回原始 HSI 网格时仍接近单位增益。

## 当前真实数据结果

### 3DSSZ

- SWIR→RGB：28 个控制点，选择 0.5 倍局部场；
- NIR→RGB：选择 0；
- NIR→SWIR：选择 0；
- 共同 HSI→RGB：选择 0；
- NIR–RGB：0.7671；
- SWIR–RGB：0.7900；
- NIR–SWIR：0.7184。

### ZKH3

- SWIR→RGB：选择 0；
- NIR→RGB：55 个控制点，选择 1.0 倍局部场；
- NIR→SWIR：73 个重叠波段控制点，选择 1.0 倍局部场；
- 共同 HSI→RGB：选择 0；
- NIR–RGB：0.4507；
- SWIR–RGB：0.5501；
- NIR–SWIR：0.6388。
