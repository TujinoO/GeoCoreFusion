# GeoCoreFusion V6.1 视觉全细节融合与验证

## 1. 版本定位

V6.1 回到 V6 的直观主线：在最新配准前端通过内部结构审计后，从 RGB 提取尽可能完整但经过保边降噪的空间细节，再注入 NIR/SWIR。V6.1 只输出 `visual_full_detail` 单产品，不调用 `scientific_conditional` 或 V8 条件门控。

接受进入产品的内容包括共同结构、RGB 独有纹理、阴影、反光、彩色标记和标签。方法只在注入前控制可避免的随机噪声、无效边界和明显配准残差。

## 2. 配准前端

1. 统一像素中心和端点坐标约定。
2. 使用跨模态结构特征完成全幅 phase/ECC/affine 粗配准。
3. 在 ROI 内执行仿射、列向低自由度推扫几何和三模态联合一致性选择。
4. 双向局部同名点采用亚像素峰值拟合，并按前后向误差、峰值 margin、搜索边界和 Jacobian 拒绝不可靠点。
5. 每个原始 NIR/SWIR 立方体只重采样一次，之后才提取和注入 RGB 细节。

受控合成真值的 P95 为：全幅粗仿射 TRE 0.231 分析像素、ROI 仿射 TRE 0.283、局部场 EPE 0.429。这证明算法组件具有亚像素能力；真实岩心 `<0.5` 原生 RGB 像素仍需独立靶标或留出同名点 TRE。

## 3. V6.1 细节链

### 3.1 RGB 保边降噪

采用强度 0.55 的双边保边降噪。随机亮度/色度噪声先被压制，裂隙、颗粒边缘、标签笔画和托盘结构保持。

### 3.2 四尺度可重建细节

尺度为 0.65、1.35、2.80、5.60 px，权重为 1.15、1.00、0.82、0.58。小尺度负责颗粒和细裂隙，中尺度负责层理和岩块边界，大尺度恢复暗区缓变对比。

### 3.3 彩色等亮边缘

同时计算亮度和色度梯度，色度权重为 0.35。逐像素选取幅值更强的候选，因此仅有颜色变化、亮度近似不变的标记或矿物边界也能进入细节场。

### 3.4 Screened-Poisson 重建

最强梯度场可能不可积。V6.1 用 screen=0.24 的 screened-Poisson 求解重建连续标量细节，避免直接叠加多尺度高通造成相位冲突、噪点和光晕。

### 3.5 注入与控制

- log 乘性细节强度 0.92；增益范围 0.52–1.92。
- 加性细节强度 0.22，用于低反射率波段和等亮彩色边缘。
- 暗区相对对比增强 0.25。
- 最终锐化 0.25，使用反射边界和 MTF-aware 限制。
- 最终产品执行 4 次低分辨率回投，权重 0.80，把实际“系数 + 增益 + 加性细节”产品循环到原始 NIR/SWIR 观测网格。

## 4. 两组真实 ROI 结果

V6/V6.1 均从保存的材料系数重建 901/1651/2201 nm 反射率，并在每个场景使用同一线性显示域。

| 场景 | 方法 | RGB/材料边界相关 | 2201 nm 可靠细节 ρ | 暗区可靠纹理 ρ | 2201 nm edge F1 | 回投 RMSE | SAM/° |
|---|---|---:|---:|---:|---:|---:|---:|
| 3DSSZ | V6 | 0.332 | 0.400 | 0.359 | 0.547 | 0.00462 | 0.923 |
| 3DSSZ | V6.1 | 0.459 | 0.734 | 0.820 | 0.626 | 0.00386 | 0.872 |
| ZKH3 | V6 | 0.246 | 0.489 | 0.416 | 0.643 | 0.01453 | 0.917 |
| ZKH3 | V6.1 | 0.324 | 0.636 | 0.582 | 0.733 | 0.00946 | 0.682 |

2201 nm RGB-flat 高频能量比从 11.13 降至 1.57（3DSSZ），从 2.39 降至 1.55（ZKH3）。这说明 V6.1 并非简单放大 V6 的颗粒状噪声，而是用更接近 RGB 的相干细节替代大量非 RGB 对齐高频。

## 5. 光谱曲线是否保留

保留的是原始尺度观测一致性。最终 HR 产品经过估计 PSF 降采样后，3DSSZ 全谱 RMSE=0.00386、SAM=0.872°、band-CC=0.99967；ZKH3 为 0.00946、0.682°、0.99834。代表性暗/中/亮像素的原始与回投光谱曲线基本重合。

新增 HR 像素仍应标记为 `RGB-textured NIR/SWIR estimate`，不能在没有独立 HR-NIR/SWIR 真值时解释为每个像素的真实 SWIR 高频光谱。

## 6. 复现

```powershell
python -m geocorefusion.cli run configs\3dssz_roi_fusion_v61.yaml
python -m geocorefusion.cli run configs\zkh3_roi_fusion_v61.yaml
python -m geocorefusion.cli run configs\3dssz_roi_fusion_v61_benchmark.yaml
python -m geocorefusion.cli run configs\zkh3_roi_fusion_v61_benchmark.yaml
python scripts\build_v61_figures.py --output-dir artifacts\v61_research\figures --summary artifacts\v61_research\experiments\benchmark_summary.json
```

正式配置不保留材料系数以节省空间；benchmark 配置仅增加系数保存，用于同一反射率显示域的科研图重建，不改变方法参数。

## 7. 当前结论

V6.1 已达到“比 V6 明显增加可见空间细节、暗区可靠纹理和边缘一致性，同时改善最终产品原尺度光谱回投”的预期。当前无需继续增大锐化或注入强度。下一步论文验证的优先项是独立配准靶标、标定斜边/暗区阶跃靶、跨钻孔盲测和局部 HR-NIR/SWIR 真值，而不是继续堆叠更复杂网络。
